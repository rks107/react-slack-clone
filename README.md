# Create a new project

make sure you have npm, node and create-react-app tool

```
create-react-app project-name
```

OR

```
git clone https://github.com/rks107/react-slack-clone.git
```

# Starting of Project

```
npm start
```

# Project Structure

```
src
├── components
│   ├── App.js
│   ├── MainContainer.js
│   ├── Sidebar.js
│   ├── SignIn.js
│   ├── Slack.js
│   └── index.js
├── firebase.js
├── index.css
├── index.js
└── providers
    └── UserProvider.js
```

# Hosting the project on Firbase

```
  npm run build
  npm i -g firebase-tools
  firebase login
  firebase deploy
```
